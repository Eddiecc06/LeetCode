# 371. Sum of Two Integers
https://leetcode.com/problems/sum-of-two-integers/description/

## Problem Description

Given two integers a and b, return the sum of the two integers without using the operators + and -.


**Example 1**:
```
Input: a = 1, b = 2
Output: 3
```
**Example 2**:
```
Input: a = 2, b = 3
Output: 5
```

**Constraints**
```
-1000 <= a, b <= 1000
```

## Solution

### _Related Topic_
   Bit Manipulation

### _C++ Code_
```cpp
class Solution {
public:
    int getSum(int a, int b) {
        
        while(b){
            unsigned c = a & b;
            a ^= b;
            b = c << 1;
        }
        return a;
    }
};
```
