# 670. Maximum Swap
[Medium](https://leetcode.com/problems/maximum-swap/description/)

## Problem Description

You are given an integer num. You can swap two digits at most once to get the maximum valued number.

Return the maximum valued number you can get.


**Example 1**:
```
Input: num = 2736
Output: 7236
Explanation: Swap the number 2 and the number 7.
```
**Example 2**:
```
Input: num = 9973
Output: 9973
Explanation: No swap.
```

**Constraints**
```
0 <= num <= 10^8
```

## Solution

### _Related Topic_
   Math, Greedy

### _C++ Code_
```cpp
class Solution {
public:
    int maximumSwap(int num) {
        string str = to_string(num);
        int n = str.size();
        vector<int> NumToIndex(10, 0);
        //The latest index of each number 
        for (int i = 0; i < str.size(); ++i)
            NumToIndex[str[i] - '0'] = i;
        for(int i = 0; i < n; ++i){
            for(int j = 9; j > str[i] - '0'; --j){
                //Find a larger number and its position is behind str[i]. Swap these two numbers 
                if(NumToIndex[j] > i){
                    swap(str[i],   str[NumToIndex[j]]);
                    return stoi(str);
                }
            }
        }
        return num;
    }
};
```
