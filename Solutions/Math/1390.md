# 1390. Four Divisors
[Medium](https://leetcode.com/problems/four-divisors/description/)

## Problem Description

Given an integer array nums, return the sum of divisors of the integers in that array that have exactly four divisors. If there is no such integer in the array, return 0.


**Example 1**:
```
Input: nums = [21,4,7]
Output: 32
Explanation: 
21 has 4 divisors: 1, 3, 7, 21
4 has 3 divisors: 1, 2, 4
7 has 2 divisors: 1, 7
The answer is the sum of divisors of 21 only.
```

## Solution

### _Related Topic_
   Array, Math

### _C++ Code_
```cpp
class Solution {
public:
    int sumFourDivisors(vector<int>& nums) {
        int ans = 0;
        for(int num : nums){
            int val = isValid(num);
            if(val != -1)
                ans += val;
        }
        return ans;
    }
    int isValid(int n){
        // the nearest integer value to the actual cube root of n.
        int p = round(cbrt(n));
        if((long long)p*p*p == n && isPrime(p))
            return 1 + p + p*p + n;
        for(int d = 2; d*d <= n; ++d){
            if(n % d == 0){
                int num1 = d, num2 = n / d;
                if(num1 != num2 && isPrime(num1) && isPrime(num2))
                    return 1 + num1 + num2 + n;
                return -1;
            }
        }
        return -1;
    }
    bool isPrime(int num) {
        if(num < 2)
            return false;
        for(int i = 2; i * i <= num; ++i)
            if(num % i == 0)
                return false;

        return true;
    }
};
```
