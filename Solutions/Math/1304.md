# 1304. Find N Unique Integers Sum up to Zero
[Medium](https://leetcode.com/problems/find-n-unique-integers-sum-up-to-zero/description/)

## Problem Description

Given an integer n, return any array containing n unique integers such that they add up to 0.

**Example 1**:
```
Input: n = 5
Output: [-7,-1,1,3,4]
Explanation: These arrays also are accepted [-5,-1,1,2,3] , [-3,-1,2,-2,4].
```
**Example 2**:
```
Input: n = 3
Output: [-1,0,1]
```
**Example 3**:
```
Input: n = 1
Output: [0]
```

**Constraints**
```
1 <= n <= 1000
```

## Solution

### _Related Topic_
   Array, Math

### _C++ Code_
```cpp
class Solution {
public:
    vector<int> sumZero(int n) {
        vector<int> ans(n, 0);
        int curNum;
        if(n % 2 == 1)
            curNum = -(n - 1) / 2;
        else
            curNum = - n / 2;

        for(int i = 0; i < n; ++i){
            ans[i] = curNum;
            ++curNum;
            if(n % 2 == 0 && curNum == 0)
                ++curNum;
        }
        return ans;

    }
};
```
